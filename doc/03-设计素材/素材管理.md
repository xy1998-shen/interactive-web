# 《楚江寻艾》素材管理

## 一、素材管理原则

本项目大量素材依赖 AI 生成，需要严格管理以确保：

1. **风格一致**：所有素材遵循同一色板和画风。
2. **可复现**：每个素材有对应的生成 prompt，可随时重新生成或微调。
3. **体积可控**：全部素材压缩后不超过 3MB。
4. **命名清晰**：文件名即可判断用途、场景和类型。

- 路径点数：每个 SVG 控制在 20-40 个锚点以内（便于变形插值）。

## 四、AI 素材生成 Prompt 库

每个素材对应一条标准化 prompt，确保可复现和风格一致。

提示词的版本管理、生成流程、质量评分和完整母版模板见 `提示词管理.md`。本章节只保留与素材规格直接相关的基础 prompt，实际生成时优先从 `提示词管理.md` 复制并记录版本。

### 通用 prompt 约束

所有 prompt 末尾统一追加：
```
--style raw --no text, watermark, signature, frame, border
```

色彩强制约束（根据需要选用）：
- 背景色：`#101C1B`（楚江青黑）
- 主体绿：`#8EA86A`（艾草青）
- 强调红：`#A43D32`（朱砂红）
- 金色：`#C8A45D`（青铜金）
- 浅色：`#E8E1D2`（雾白）

### 各素材 Prompt

**艾草精灵**
```
single mugwort artemisia branch, Chinese ink wash painting style,
transparent background, PNG with alpha channel,
semi-transparent leaves, delicate leaf veins visible,
soft green #8EA86A tones, isolated subject, artistic brush strokes, no text
--ar 1:1 --style raw
```
后处理：用 rembg 去除残余背景，确保 alpha 通道干净。

**龙舟剪影**
```
dragon boat silhouette side view, Chinese ink brush stroke style,
dark vermillion #A43D32 with gold #C8A45D accents,
minimal detail, long horizontal composition,
transparent background, isolated subject, no text
--ar 3:1 --style raw
```
后处理：去背景后检查船体边缘无杂色。

**编钟剪影**
```
ancient Chinese bronze bell bianzhong silhouette, front view,
gold #C8A45D line art, transparent background,
minimal engraving details visible, ink painting style,
vertical composition, isolated subject, no text
--ar 1:2 --style raw
```
后处理：去背景，保留金色线条的抗锯齿边缘。

**暗色江面背景**
```
dark river surface at night, top-down view,
very dark teal #101C1B with subtle ripple patterns,
Chinese ink painting atmosphere, minimal,
cream #E8E1D2 moonlight reflections, cinematic, no text
--ar 16:9 --style raw
```

**雾气层**（用作 alpha 遮罩或半透明叠加）
```
horizontal fog wisps on pure black background,
Chinese ink painting style, cream white #E8E1D2 fog,
ethereal and sparse, horizontal composition,
layered depth, no text
--ar 16:9 --style raw
```
说明：此素材保留黑底是有意的——黑底部分在 PixiJS 中作为透明区域使用（通过混合模式 screen 或作为 alpha mask 的灰度源）。不需要去背景，但需确保雾气部分为纯白/浅色，黑底部分为纯黑，中间过渡干净无杂色。

**水纹纹理（可平铺）**
```

subtle concentric ripple patterns, very dark teal #101C1B,
ink painting style, top-down view, no text

```

## 五、素材生成后处理流程

每个 AI 生成的素材必须经过以下步骤才能进入项目：

```
步骤 1：生成多个变体
  → 每个素材生成 4-6 张变体，选择最贴合风格的

步骤 2：去背景（精灵类）
  → 使用 rembg 或 Photoshop 去除背景
  → 确保边缘干净无杂色

步骤 3：色彩校正
  → 对照色板调整色阶和曲线
  → 确保主色调落在 5 色系统范围内
  → 去除不属于色板的杂色

步骤 4：尺寸调整
  → 按规格表裁切/缩放到目标尺寸
  → 精灵类保持 2 的幂次尺寸（512/1024）利于 GPU

步骤 5：格式转换与压缩
  → 精灵/背景 → WebP（质量 80-90）
  → 噪声/置换图 → PNG 灰度 8bit
  → 矢量 → SVG（手动简化锚点）

步骤 6：命名与归档
  → 按命名规范放入对应目录
  → 更新素材清单表

步骤 7：效果验证
  → 在 PixiJS 测试场景中加载验证显示效果
  → 确认透明度、混合模式、缩放表现正常
```

### 问诗场景预设诗句（待补充完整）

选字池：江、艾、舟、鼓、风、月、青、钟

**匹配规则（确定性，禁止随机）**：
1. 将用户选中的 3 字按固定字典序排列生成 key（如选"风、江、艾"→ key 为"艾江风"）

3. 未命中 → 找"包含选中 3 字中任意 2 字"的首条匹配
4. 仍未命中 → 使用第一个选中字的固定兜底句

**精确匹配预设（示例，需补充至 20 首）**：

## 十、一句话素材准则

所有素材必须像"从同一卷水墨长卷中裁下的碎片"：色调统一、笔触一致、意象相关，拼在一起即成完整画面。
