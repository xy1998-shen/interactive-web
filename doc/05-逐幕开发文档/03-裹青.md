# 03 裹青

## 场景描述

上一幕寻艾结束后，清晨村户的生活感继续向屋内推进。第三幕进入临水屋舍：旧木桌上有箬叶、糯米、竹篮、青线和蒸煮器物，人物围坐包粽、等待、分食，窗外仍有雾青江光。

当前实现不再使用抽象江面和大量漂浮粒子，而是以四张生活场景面板顺序展开：备料、裹粽、蒸煮、分食。用户看到的是端午裹粽如何从材料准备走向一家人的节日餐桌。

标题层与第二幕对齐：左上保留 `03 / 楚江寻艾` 与大标题 `裹青`，不显示额外副标题，避免遮挡画面和知识卡。

## 目的

- 把第二幕“艾草入户”的生活空间继续向内推进，形成“寻艾入户 → 桌前裹青 → 江上听鼓”的线索。
- 用场景化动画表现端午手作：箬叶、水纹、青线、蒸汽、桌面暖光，而不是用不贴合场景的大量抽象粒子。
- 保持节奏自动推进：四个面板只按顺序播放一次，到最后一面停住，不循环。
- 在完成态自动展示知识卡，让用户完整看到“端午裹青 · 粽俗拾趣”，不需要点击展开。

## 交互

1. 入场显示第一面“备料”：桌面材料铺开，叶影、水纹和米粒微光轻动。
2. 系统自动顺序切到“裹粽”：青线弧线、叶片压影和桌面阴影表达手作收束。
3. 继续切到“蒸煮”：蒸汽曲线、锅具暖光和轻微热雾表现等待。
4. 最后切到“分食”：桌面暖光、香气线和人物围坐形成完成态。
5. 到达第四面后约 3 秒自动弹出知识卡，知识卡默认完整展开，不运行收缩逻辑。
6. 底部不再显示“轻点画面，开始裹青”等提示文案，只保留面板指示点和完成后的下一幕推进。

面板指示点允许用户手动切换已加载的场景，但自动播放逻辑不会从第四面回到第一面。

## 技术

- 场景类为 `js/scenes/wrap-scene.js`，继承 `BaseScene` 的面板能力。
- `initPanels()` 定义四个面板：`prep`、`wrap`、`steam`、`eat`。
- `startAutoPlay()` 负责顺序推进；公共逻辑已移除循环，到最后一个面板后停止。
- 每个面板使用贴合物象的 Pixi Graphics / Sprite 动效：
  - `buildPanel1Ripples()`：桌面水纹、叶影、米粒微光。
  - `buildPanel2Threads()`：青线弧线、叶片压影、收束阴影。
  - `buildPanel3Steam()`：蒸汽曲线、锅口暖光。
  - `buildPanel4Glow()`：桌面暖光、香气线、少量节日氛围光。
- `scheduleWrapKnowledge()` 在第四面停留后调用 DOM 层 `showKnowledgeCard()`。
- 右下知识卡由 `js/dom-layer.js` 与 `css/style.css` 控制，默认 expanded，关闭折叠箭头与收缩交互。

## 文案

### 面板文案

1. 备料：箬叶洗净，糯米浸透。端午前夕，家家户户忙碌着同一件事。
2. 裹粽：两片叶交叠成斗，米实馅满，丝线一缠。手艺在指尖代代相传。
3. 蒸煮：文火慢煮数时辰，粽香渐浓。等待，也是节日仪式的一部分。
4. 分食：剥开青叶，糯香扑鼻。孩子的笑脸，是端午最好的注脚。

### 知识卡

- 标题：端午裹青 · 粽俗拾趣
- 要点：
  - 为何端午裹青：青色粽叶寓意生机，取“青”字谐音“清”，象征驱毒辟邪、洁净迎夏。
  - 各地风俗差异：北方黄米红枣甜粽，江浙鲜肉蛋黄咸粽，闽粤碱水粽、烧肉粽等。
  - 文化延续：从角黍投江的传说，到家家户户的手作粽，裹粽把纪念、祈福和团圆包进一片青叶。

第三幕不再使用底部固定提示句“轻点桌上青叶，把糯米与诗句收成粽形。”，标题区也不显示副标题。

## 验收要点

- 能明确看出四个生活场景：备料、裹粽、蒸煮、分食。
- 第三幕动效应贴合物象，不出现大面积漂浮粒子、抽象星点或不真实的光效堆叠。
- 四个面板按顺序自动播放一次，到第四面后停止，不循环回第一面。
- 第四面停留后知识卡自动弹出并完整展开，不需要用户点击展开，也不执行收缩。
- 左上标题层与第二幕对齐，包含幕号、项目短题和大标题；无副标题文案。
- 底部不再保留“开始裹青”提示胶囊，避免与面板指示点和知识卡冲突。
- 完成态能自然引出第四幕“听鼓”，保留端午节日氛围与江岸生活感。
