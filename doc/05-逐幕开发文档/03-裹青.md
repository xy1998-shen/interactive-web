# 03 裹青

## 场景描述

画面中央是两片展开的粽叶，像一张可折叠的青色纸形。米粒散落在叶面和浅水反光之间，少量文字粒像米粒一样漂浮。背景延续雾青水面，但比前两幕更集中、更有手作感。

## 目的

- 把包粽子的日常动作抽象为“把祝愿包入端午”。
- 用固定三段自动演出控制复杂度，避免做真实包粽教程。
- 让艾草轨迹自然过渡为粽叶折线。

## 交互

1. 用户轻点画面中央或粽叶光点，启动自动裹青演出。
2. 左叶先向中心旋转合拢，米粒和文字粒第一次聚拢。
3. 右叶继续向中心旋转合拢，细线从画面下方浮出并自动横向掠过粽形。
4. 系统完成缠绕收束，形成稳定三角粽轮廓。

播放前目标点轻微呼吸提示，播放过程中不需要用户连续操作。完成后米粒和文字粒聚合到粽形内部。

## 技术

- 两片粽叶作为独立 Pixi Sprite，锚点设置在叶尖或折叠轴附近。
- 每一步有独立状态：`started`、`leftFolded`、`rightFolded`、`tied`。
- 触发 hit area 大于视觉光点，降低点击难度。
- 折叠用 GSAP 旋转和位移，不做自由形变。
- 米粒、字粒建议用 Pixi Graphics / Text 动态生成。
- 自动演出期间禁用重复点击，完成后再出现下一幕入口。

## 文案

- 短句：青叶合拢，江心有愿。
- 轻知识：粽子承载纪念与寄情，也是端午最日常的符号。

## 验收要点

- 交互是轻点触发的固定自动流程，不能让用户自由乱折。
- 播放前提示清楚，播放中节奏稳定，不出现失败惩罚。
- 完成态必须像粽形，而不是两片普通叶子叠在一起。
- 能自然过渡到龙舟或鼓点线索。
