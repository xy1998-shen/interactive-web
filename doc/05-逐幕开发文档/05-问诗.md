# 05 问诗

## 场景描述

画面是一片由雾青江面转入深青静水的安静水面。水上漂浮 8 个中文意象字粒：江、艾、舟、鼓、风、月、青、钟。远处有极淡竹简或古卷暗影，整体温和、沉静，不阴森。

## 目的

- 把楚辞气韵转化为水中文字的生成体验。
- 用轻量诗意承接屈原文化和端午追思。
- 让用户从鼓点高潮进入情绪沉淀。

## 交互

- 用户从 8 个字粒中选择任意 3 个。
- 被选中字粒亮起并向上漂浮。
- 第 3 个字粒被选中后，系统锁定结果。
- 根据所选 3 字从预设短句库中匹配 1 条原创短诗。
- 短诗以纵向或水纹浮字形式出现。

## 技术

- 字粒使用 Pixi Text 或 DOM，保持清晰可读。
- 字粒位置固定或轻微正弦漂浮，避免遮挡。
- 匹配逻辑必须确定性：同一组选字永远返回同一首诗。
- 生成 key 时将 3 字按固定字典序排列。
- 精确命中用对应诗句；未命中则找包含 2 字的首条；仍未命中使用第一个选中字的兜底句。
- 禁止接入在线随机生成，避免不可控文案。

## 文案

- 字粒：江、艾、舟、鼓、风、月、青、钟。
- 短句：字从水起，风过楚江。
- 轻知识：屈原诗意与端午记忆相连，是楚江旅程的精神底色。
- 示例短诗：江风拂艾，五月入舟。

## 验收要点

- 8 个字必须清楚可读。
- 选满 3 个后立即锁定，不继续追加文字。
- 短诗不铺满屏幕，只显示一条结果。
- 语气温和诗意，不出现阴郁或生硬百科表达。
