# 06 和鸣

## 场景描述

深青江面上方或中景出现抽象青铜编钟剪影。用户触碰后，青铜金圆形声波从钟影扩散到江面，前几幕的艾叶、粽叶、龙舟、诗字以低透明度短暂回归。最终声波开始收束，形成尾声端午印记。

## 目的

- 用编钟意象完成荆楚文化的升华。
- 把前面分散的视觉和声音线索合成一次安静的收束。
- 不扩展百科信息，只做审美和情绪上的合奏。

## 交互

- 用户轻触钟影或声波中心。
- 青铜金声波向外扩散。
- 江面、水纹、艾叶、龙舟、诗字短暂叠合。
- 出现短句“金声入水，楚江和鸣。”
- 声波收束后进入尾声。

## 技术

- 编钟使用 2D 剪影，不做复杂 3D。
- 声波可用 SVG 圆环或 Pixi Graphics 多层圆环实现。
- 扩散时长应慢于普通按钮点击波纹，建议 1200ms 至 1800ms。
- 闪回素材透明度控制在 10% 至 18%。
- 多层素材需要统一混合模式，避免画面脏或过亮。

## 文案

- 短句：金声入水，楚江和鸣。
- 轻知识：编钟礼乐是荆楚文化记忆，与端午江声彼此呼应。

## 验收要点

- 声波扩散要开阔、慢、稳，不能像普通按钮反馈。
- 编钟不能像博物馆展陈截图，要融入江面。
- 闪回素材只能做回忆，不抢编钟和声波主视觉。
- 完成后能自然形成端午印记。
