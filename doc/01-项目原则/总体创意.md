# 《楚江寻艾》总体创意

## 一、创意定位

《楚江寻艾》是一件以湖北屈原故里端午习俗为文化底色的沉浸式交互网页作品。

作品不做传统的非遗资料展示站，而是把端午习俗转化为一段可触摸、可聆听、可推进的数字仪式。用户沿着楚江进入端午现场，通过拨雾、寻艾、折粽、击鼓、行舟、触钟等动作，逐步感受屈原故里端午的诗意、礼俗与荆楚文化气质。

核心表达是：让用户不是“阅读端午”，而是“亲手经过端午”。

## 二、作品目的

《楚江寻艾》希望用户在浏览结束后获得的不只是“知道端午有什么习俗”，而是对湖北屈原故里端午形成一段有画面、有动作、有情绪的记忆。

用户完成体验后，应当获得以下收获：

1. **认知收获**：知道端午不只是粽子和假日，也包含悬艾、龙舟、诗意纪念、地方礼俗等更丰富的文化内容。
2. **地域收获**：意识到湖北与端午、屈原故里、荆楚文化之间的关系，感受到“楚江端午”的地方辨识度。
3. **审美收获**：通过江雾、艾草、粽叶、龙舟、楚辞文字和编钟声波，感受一种温和、克制、沉浸的东方诗意。
4. **互动收获**：通过拨雾、寻艾、折粽、击鼓、触字、听钟等动作，理解传统文化可以被转化为可参与的数字体验。
5. **情绪收获**：在安静、渐进的体验中获得对端午的亲近感，而不是把非遗视为遥远、严肃或只能被讲解的内容。

为避免体验结束后只留下“画面很好看”的印象，作品会在每幕完成后解锁一句轻知识，并在尾声通过端午印记进行汇总。用户最终应能带走四个明确记忆点：

- **识一味清艾**：端午悬艾承载祈安、避疫、迎夏的生活愿望。
- **裹一叶青愿**：粽子不只是食物，也是纪念、寄情和日常团圆的符号。
- **听一声江鼓**：龙舟竞渡体现江上协作、节庆动员和湖北水乡气质。
- **记一段楚风**：屈原诗意与荆楚礼乐共同构成湖北端午的精神底色。

作品最终希望解决的问题是：如何让年轻用户在短时间内愿意进入传统文化，并通过亲手操作留下清晰记忆点。

## 三、主题名称

主标题：**楚江寻艾**

副标题：**一场屈原故里端午的沉浸式交互诗**

标题选择“寻艾”，是因为艾草是端午中温和、清晰、具有识别度的意象，既能承接驱疫祈安的民俗含义，也能转化为视觉中的叶影、香气、纹样和交互线索。“楚江”则指向湖北地域、荆楚文化和江上端午场景。

## 四、体验概念

作品采用单页分幕式体验，不依赖外部链接，不做普通导航菜单。

用户进入页面后，面对一片明亮的雾青江面、远山云气与小舟悬念。通过第一次轻点，水纹被拨开，端午旅程开始。此后，一条弯曲的江流作为隐性的章节路径，带出不同场景：

1. **入江**：拨开江雾，进入楚江端午的时间与空间。
2. **寻艾**：江边村庄采艾归来，窗前悬艾守住盛夏门户。
3. **裹青**：折叠粽叶，让米粒与诗句在叶面中聚合。
4. **听鼓**：按节奏击鼓，推动龙舟穿过江面。
5. **问诗**：触碰水中浮现的文字，生成短句式楚辞意象。
6. **和鸣**：轻触编钟光影，完成江面、龙舟、艾草、诗句的收束。

每一幕只保留一个主要触发动作，触发后由自动演出完成本幕，避免功能堆叠，让交互更像仪式，而不是小游戏集合。

## 五、视觉风格

整体风格定位为：**诗意东方 + 实验互动**。

视觉上参考水墨、壁画、卷轴、青铜器和端午民俗色彩，但不直接复刻传统插画或古风页面。画面应保持克制、留白和电影感，以慢动效和材质层次制造沉浸感。

建议色彩：

- 晨雾青、浅水青：用于首屏江面、薄雾亮部和开阔感。
- 云水米白、雾白：用于远山云气、文字、呼吸感动效。
- 楚江青黑、深墨绿：用于深水、阴影和后段层次。
- 艾草青：用于核心线索和生命感。
- 朱砂红：用于龙舟、鼓点、节庆强调。
- 青铜金、夕照暖金：用于编钟、章节节点、天光和收束光效。

画面关键词：江雾、水纹、艾叶、粽叶、红绸、龙舟剪影、楚辞文字、编钟光影。

## 六、交互风格

本作品的交互重点不是复杂编程，而是让每个动作都与主题相关。

参考方向：

- 学习“章节路径式”的沉浸导航，让用户沿一条江流推进，而不是点击普通菜单。
- 学习“开卷式”的进入仪式，但转化为拨开江雾、展开水纹。
- 学习“工艺步骤可视化”，但去掉说明书式菜单，把材料变成可触摸对象。
- 学习“电影化东方视觉”，用少文字、大氛围、慢节奏和声音暗示建立高级感。

交互动作应尽量自然：

- 轻点与点击：拨雾、悬艾入户、自动裹青、击鼓、触字；寻艾围绕“把艾草挂到窗前或门楣”完成。
- 自动演出：每幕触发后通过慢动效、轨迹和光点完成叙事，不要求用户持续拖动。
- 点击节奏：模拟鼓点，推动龙舟。
- 轻触：触发编钟声与最终光效。

## 七、作品亮点

1. **地域明确**：以湖北屈原故里端午习俗为基础，避免泛化的端午主题。
2. **形式原创**：不是信息介绍页，而是“端午仪式”的交互化表达。
3. **审美统一**：江、艾、舟、诗、钟构成连续的视觉和叙事线索。
4. **实现可控**：每幕只做一个核心动作，适合在网页中完成并打磨细节。
5. **符合比赛约束**：单页本地作品，不依赖外部跳转，所有页面与主题一致。

## 八、暂定一句话方案

《楚江寻艾》以湖北屈原故里端午习俗为灵感，把艾草、粽叶、龙舟、楚辞与编钟转化为可触摸的数字意象，让用户沿楚江完成一场诗意、温和、沉浸的端午交互旅程。
